image labeler

Welcome to image labeler. It is exactly what it sounds like.

  • Inspired by Google Image Labeler
  • Join a partner and rack up points by labeling images with the same label as your partner
  • You have 30 seconds per image, and you can enter as many labels as you want
  • The longer the label (in words), the more points you get
  • Built with node.js and socket.io. View the project on Github
